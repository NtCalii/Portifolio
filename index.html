Tecnologias

Python: Uma linguagem de programação versátil e amplamente usada em desenvolvimento web, automação, ciência de dados e mais. É conhecida por sua sintaxe clara e pela vasta quantidade de bibliotecas.

Django: Um framework de alto nível para Python, focado em desenvolvimento rápido e seguro de aplicações web. Ele segue o padrão MVC (Model-View-Controller).

MySQL: Um sistema de gerenciamento de banco de dados relacional, popular por sua eficiência e confiabilidade. É amplamente usado para armazenar e gerenciar dados em aplicações web.

SQLite: Um banco de dados relacional leve, ideal para aplicações menores ou locais. Diferente de outros, ele não requer um servidor dedicado, armazenando dados em um único arquivo. É amplamente usado em aplicações web, mobile e desktop.

HTML: A base da estrutura de páginas web, mostrando o conteúdo na web usando tags como (div, p e img). Ele trouxe recursos avançados como suporte nativo a vídeo, áudio e tags semânticos, criando sites mais interativos.

CSS: Ele faz a estilização visual de páginas web, controlando cores, fontes, layouts e animações. Têm recursos para fazer layouts responsivos, além de transições e transformações.

JavaScript (JS): Uma linguagem de programação essencial para interatividade no lado do cliente. É usada para manipular o DOM, criar animações e lidar com eventos em páginas web, além de ser a base para frameworks como React e Vue.js.

GitHub: Uma plataforma para controle de versão e colaboração em projetos. Usa o sistema Git para gerenciar código, permitindo que equipes trabalhem juntas, acompanhem mudanças e mantenham os código organizados.
